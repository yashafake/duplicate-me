# Security Policy

## Supported Versions

This project is in active early development. Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Please avoid opening public issues for security vulnerabilities.

Preferred channel:

- GitHub Security Advisories (private report) for this repository

Fallback channel:

- Email: radchenko.yakov@gmail.com

Include:

- affected version or commit
- reproduction steps
- impact assessment
- potential mitigation ideas (if available)

You will receive an acknowledgment as soon as possible, and we will coordinate disclosure responsibly.
