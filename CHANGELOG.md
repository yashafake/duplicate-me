# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning where practical.

## [0.1.0] - 2026-04-21

### Added

- Initial public release of DuplicateMe.
- Swift core modules for scan orchestration, persistence, and media fingerprinting.
- CLI workflows for scanning, rescanning, exporting reports, ignore rules, and trash actions.
- Local review server and browser-based review interface.
- Electron desktop shell that embeds the local review UI.
